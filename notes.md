# Week 1 - Day 1 Learnings

In Day 1, I learned how to:

- Set up a Python notebook environment for LLM engineering
- Load environment variables securely using a `.env` file
- Read the OpenAI API key without hardcoding secrets in the notebook
- Create a simple OpenAI client using the Python SDK
- Send a basic message to an LLM model and read the response
- Understand the structure of `messages` with `role` and `content`
- Fetch website content using a scraper utility
- Create system prompts and user prompts
- Build a simple website summarizer using an LLM
- Display LLM-generated summaries using Markdown
- Experiment with prompt tone, such as a humorous or snarky summary style

Important: I did not upload my `.env` file or API keys to GitHub.
